# Claude Mistakes & User Rules — Say My Name Vapes

This file must be reviewed at the start of every session on this project.

---

## ❌ DO NOT DO THESE

### 1. No self-made logos
**Do not** create a logo using SVG, text, or any code. The logo is the owner's asset.
**Do:** Place a clearly marked logo placeholder (`[LOGO]` box or `<img>` tag with a fallback) where the logo should go.

### 2. No Heisenberg silhouette or Breaking Bad references
**Do not** draw Heisenberg, the porkpie hat, the glasses, the goatee, or anything from Breaking Bad.
**Do not** use the hat+glasses SVG, no Heisenberg holographic figure, nothing.
This was done 3 times and rejected every time.

### 3. No Breaking Bad / meth-lab themed design
**Do not** use dark navy (#06080f) + blue glow + smoke atmosphere to create a "lab" aesthetic.
**Do not** use copy like "The Quantum Chemistry", "Est. Albuquerque", "Find the Lab".
**Do:** Design a clean, modern, professional vape store — think premium retail, not TV show.

### 4. No text-as-logo in the navbar
**Do not** put "SAY MY NAME" in Bebas Neue + "Vapes" in script font as a substitute logo.
**Do:** Put the store name as text alongside a real logo placeholder image/box.

---

## ✅ WHAT THE USER WANTS

- **Clean vape store UI** — professional, modern, minimal
- **Logo placeholder** clearly marked where the real logo goes
- **Store name** displayed properly as text
- **Nice pictures** — good lifestyle/product imagery or high-quality placeholders
- **Fresh color palette** — nothing from the old Breaking Bad theme
- **Standard vape shop navigation** — products, about, contact etc.
